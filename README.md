# my_os

**my_os** — это минималистичная операционная система, разрабатываемая на языке Rust в рамках дипломной работы бакалавра.

Основной целью проекта является изучение низкоуровневого программирования, архитектуры компьютеров и особенностей системного программирования на Rust. Проект вдохновлён работами Филиппа Оппермана ([Philipp Oppermann's Blog](https://os.phil-opp.com/)).
